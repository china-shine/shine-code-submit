shine-code-submit 新机器部署说明

前置：机器上需要先有 Node.js（下一步 npm install -g bun 要用到）。没有的话先去 https://nodejs.org 装一个。

1，安装 bun

npm install -g bun

装完务必重开一个新终端（让 PATH 生效），再跑一下 bun --version，确认能输出版本号。

2，打开 claude，在 claude 里依次录入下面两行（注意每行左侧开端不要有空格）：

/plugin marketplace add http://8.130.168.121:9001/AI/shine-code-submit.git

/plugin install shine-code-submit@shine-code-submit

选择安装到userscope

Install for you (user scope)


3，安装完成后，退出 claude，到一个项目目录里重新打开 claude。正常的话会直接弹出网页，用来显示当前项目的记录。

如果开新会话后没任何反应（既没弹网页、也没看到链接）：在终端分别跑 bun --version 和 node --version，确认两个都能输出版本号（说明都在 PATH 里）；确认后重开 claude 再试一次。
